## 書籍の内容について

- Pythonで面白いアプリを作りたいという人に
- Webアプリならどこからでも利用可能
- 画像SNS、ラーメン判定、対戦型ゲームなど楽しいアプリが満載
- 機械学習・深層学習やChatGPT APIも!
- フレームワークやライブラリを使って簡単作成
- セキュリティ対策や2要素認証も解説
